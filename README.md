# Projeto Integrador - Sistema de Vendas

##  Descrição

Sistema web desenvolvido para controle de vendas, movimentações e boletos.

##  Tecnologias utilizadas

* Java
* JSP / Servlets
* MySQL
* Apache Tomcat

## Vídeo de apresentação

Assista ao vídeo:
(https://youtu.be/VQDvLV9yGPM)

## Funcionalidades

* Cadastro de produtos
* Movimentação de estoque
* Controle de boletos
* Marcar como pago
* Alertas de vencimento
